# java-lotto-precourse

## 구현할 기능 목록

1. 입력
- [x] 로또 구매 금액을 입력받는다
  - [예외] 천원으로 나누어 떨어지지 않으면 예외이므로 다시 입력 받는다
  - [예외] 음수이면 예외이므로 다시 입력 받는다
  - [예외] 정수가 아니면 예외이므로 다시 입력 받는다
- [x] 당첨 번호를 입력받는다
  - 당첨 번호는 쉼표로 구분된다
  - [예외] 중복된 숫자가 존재하면 예외이므로 다시 입력 받는다
  - [예외] 당첨 번호가 1 ~ 45 사이의 값이 아니면 예외이므로 다시 입력 받는다
  - [예외] 당첨 번호가 6개가 아니면 예외이므로 다시 입력 받는다
- [x] 보너스 번호를 입력받는다
  - [예외] 보너스 번호는 당첨번호와 같다면 예외이므로 다시 입력 받는다
  - [예외] 당첨 번호에 이미 존재하는 번호면 예외이므로 다시 입력 받는다
  - [예외] 당첨 번호가 1 ~ 45 사이의 값이 아니면 예외이므로 다시 입력 받는다

2. 비즈니스 로직
- [x] 금액만큼 로또를 발행한다
    - 로또를 발행할 때마다 6개의 로또 번호를 결정해야 한다
- [x] 당첨 번호와 보너스 번호를 토대로 각 로또의 당첨 여부를 판정한다
  - 로또 번호와 보너스 번호는 중복되지 않아야 한다
  - 일치하는 번호의 개수에 따라서 1등 ~ 5등, 낙첨으로 결정된다
- [x] 각 등수별 당첨 현황과 총 수익률을 포함한 당첨 통계를 도출한다
  - 각 등수별로 당첨 조건, 당첨 금액, 당첨된 복권 수를 계산해야 한다
  - 수익률은 (총 당첨액) / (총 구매액)이고 소숫점 둘째자리까지 계산한다

3. 출력
- [x] 당첨 통계를 출력한다
- [ ] 예외 발생 시 에러 문구를 출력한다
  - 문구는 "[ERROR]"로 시작해야 한다
